# Horizon 2035 — Cinematic Presentation

Premium Next.js slide deck with **Three.js** particle network, glass morphism UI, and Framer Motion transitions.

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) → press **F** for fullscreen.

## Controls

| Key | Action |
|-----|--------|
| `→` / `Space` | Next slide |
| `←` | Previous |
| `F` | Fullscreen |
| `O` | Overview grid |

## Stack

- **Next.js 16** · **React Three Fiber** · **drei**
- **Framer Motion** · **Tailwind CSS v4**
- **Instrument Serif** + **Inter** + **JetBrains Mono**
